Avatar

An avatar is a visual representation of a user or entity.

Circular Profile Avatar! ;

Circular Profile avatar could be created using avatar class, inside img tag. 
                <img
                  class="avatar avatar-cicular sm"
                  src="https://i.pravatar.cc/60"
                  alt="avatar-circular-1"
                />
                <img
                  class="avatar avatar-cicular md"
                  src="https://i.pravatar.cc/100"
                  alt="avatar-circular-2"
                />
                <img
                  class="avatar avatar-cicular lg"
                  src="https://i.pravatar.cc/150"
                  alt="avatar-circular-3"
                />
